# Анализ поведения пользователей приложения магазина продажи продуктов питания  #

**Целевая задача Заказчика**
Оценка влияния изменения шрифтов в мобильном приложении стартапа по продаже продуктов питания на поведение пользователей мобильного приложения
В рамках реализации целевой задачи пользователей приложения разбили на 3 группы: 2 контрольные со старыми шрифтами и одну экспериментальную — с новыми, для последующего проведения А/А/В-тестирования, необходимого для принятия решения о возможности смены шрифтов без потери количества пользователей
  
**Цели исследования**
- анализ воронки событий - поведение пользователя мобильного приложения; выявление того, на каком этапе существенно падает конверсия пользователей
- оценка влияния результатов А/В-тестирования гипотезы о смене шрифтов на поведение пользователей

**Задачи исследования**
- формирование воронки событий
- оценка конверсии пользователей по воронке
- проверка корректности формирования групп для А/В-тестирования
- расчет конверсии пользователей по воронке событий

**Резюме - рекомендации**
1.  Анализ изменения количества пользователей по воронке событий как по всем пользователям, так и с учетом последовательного открытия пользователем всех страниц, показал, что
   - больше всего теряется пользователей на этапе перехода с главной страницы (MainScreenAppear) на страницу выбора товара в каталоге (OffersScreenAppear) - 38,1% (по всем) и 45% (с учетом прохождения всех шагов) пользователей
   - до шага оплаты товаров из корзины (PaymentScreenSuccessful) доходит 47,7% всех пользователей, тогда как <b>среди пользователей, прошедших все шаги воронки - <u>от главной страницы, до оплаты товара доходит только 6% пользователей
2. По результатам A/A тестирования определено - А/А-тест прошел успешно:
  - различие в количестве пользователей по группам составляет менее 1%, и оно статистически не значимо
  - различие в количестве событий между контрольными группами составляет менее 1%
  - проверка статистической значимости разницы долей пользователей совершивших одно и то же событие между контрольными группами А1 и А2 по всем событиям "воронки событий" показала, что разница долей между группами статистически НЕ значима и нулевая гипотеза о равенстве долей не может быть отвергнута
3. В результате проведения A/В-тестов разницы долей пользователей совершивших одно и то же событие между контрольными группами (как по каждой, так и по объединенной) и экспериментальной группой B по всем событиям "воронки событий" показала, что разница долей между группами статистически НЕ значима и нулевая гипотеза о равенстве долей не может быть отвергнута
Таким образом, можно утверждать, что на поведение пользователей изменение шрифта значимого эффекта не оказало.
