# Анализ данных компмании-разработчика widget#

**Цель исследования**
- выявление характерных признаков потребления
- выявление профиля потребителя по сегментам
- разработка рекомендаций для компании-разработчика widget по сбору отзывов

**Задачи исследования**
   1. определение и анализ структуры и динамики ключевых метрик в разрезе пользователей и платежей
   2. анализ пользовательских тарифов и виджетов
   3. сегментация покупателей и описание характерных черт покупателей различных сегментов
   4. анализ конверсии и жизненной ценности покупателей в целом и в разрезе наиболее популярных тарифов
   5. проверка гипотез 

**Резюме - рекомендации клиенту**
1. реализация мероприятий, направленных на удержание пользователей, не осуществляющих подключение платного тарифа после бесплатного периода: определение причин отказа от продолжения использования, персональные предложения до окончания срока бесплатного подключения
2. реализация мероприятий по работе с целевой аудиторией – клиенты, у которых есть потребность в формировании 2х и более виджетов
3. анализ причин отмены платежей и заказов, реализация мероприятий, направленных на сокращение доли отказов 

**Бибилиотеки, использованные в проекте**
- pandas
- numpy
- scipy
- matplotlib
- seaborn
