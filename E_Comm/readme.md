# Анализ профиля потребителей интернет-магазина #

**Целевая задача Заказчика**
Увеличение продаж и прибыли интернет-магазина «Пока все еще тут», за счет:
- повышения эффективности маркетинговых мероприятий
- обеспечения индивидуального подхода к потребителю
  
**Цели исследования**
- выявление характерных признаков потребления 
- выявление профиля потребителя по сегментам
- выявление проблемных точек и потенциальных точек роста

**Задачи исследования**
- определение и анализ распределения и сезонности ключевых метрик: 
- количество и ценовые диапазоны приобретаемых товаров и товарных позиций, «типовая» потребительская корзина
- количество и давность заказов, характерность повторных покупок
- финансовые показатели по покупателям: выручка от продаж, средний чек
- осуществить сегментацию покупателей, описать потребительский профиль

**Резюме - рекомендации**
1. Потенциальным направлением роста выручки по TOP-сегменту покупателей являются продажа товаров из высоких ценовых диапазонов, к которым относятся товары для спальни, сумки-тележки и товары для уборки и ремонта
2. Потенциальным направлением роста выручки по MIDDLE-сегменту покупателей являются продажа товаров для спальни, ремонта и ванной, обороты по которым довольно низкие, а средний чек находится в среднем ценовом диапазоне
