# Анализ  рынка заведенийобщественного питания Москвы 2022 #

**Целевая задача Заказчика**
Открытие кафе в Москве, для чего ему необходимо:
- получить аналитику по рынку заведений общественного питания Москвы, которая позволит сформировать необходимую базу для принятия решения об открытии
- получить детализацию характеристик рынка в категории московских кофеен
  
**Цели исследования**
- выявление основных характеристик рынка заведений общественного питания 
- выявление ключевых характеристик кофеен Москвы

**Задачи исследования**
- анализ структуры рынка заведений общественного питания по категориям и географическим предпочтения
- анализ распределения заведений по среднему чеку и рейтингу
- анализ распределения и характеристик кофеен по округам Москвы

**Резюме - рекомендации**
1. Для открытия кофейни стоит рассматривать те округá Москвы, в которых еще не переполнен рынок: СЗАО, ЮВАО или ЮЗАО
2. Если в качестве фактора, определяющего целевое местоположение кофейни добавить максимальную среднюю стоимость чашки кофе, которая влияет на целевые показатели прибыльности проекта, то среди трех вышеперечисленных локаций предпочтение стоит в первую очередь остановить на ЮЗАО
3. Третье место среди округов по рейтингу кофеен, также является дополнительным положительным фактором для рассмотрения целевой локации кофейни в ЮЗАО
4. С учетом результатов анализа наиболее популярного графика работы кофеен, в случае если в качестве целевой будет определена локация в ЮЗАО, то стоит рассмотреть возможность работы кофейни по графику «ежедневно и круглосуточно», т.к. в данном округе их совсем незначительное количество и такой график может стать конкурентным преимуществом
5. Рекомендация по открытию кофейни
- целевая локация – ЮЗАО
- целевая средняя цена капучино - 184,2 руб.
- преимущественный график работы заведения - "ежедневно и круглосуточно«
- присоединение к действующим или создание новой сети - не важно, т.к. доля на рынке сетевых и несетевых кофеен равнозначна
