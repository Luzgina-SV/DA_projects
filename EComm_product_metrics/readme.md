# Анализ данных пользователей приложения #
 
**Цели исследования**

Определение причин неэффективности и разработка рекомендаций по повышению эффективность процесса привлечения новых пользователей

**Задачи исследования**
- анализ удержания и конверсии пользователей и динамики их изменения.
- анализ общей окупаемости рекламы, а таже в аналитике по странам, устройствам и рекламным каналам
- выявление ключевых проблем окупаемости

**Резюме**
1. Основываясь на том, что основная доля пользователей из США, где основными и, как оказалось, самым неэффективными каналами привлечения являются TipTop и FaceBoom, необходимо пересмотреть работу с этими каналами: оптимизировать рекламный бюджет или изменить рекламную кампанию, чтобы повысить «доходность» на одного клиента. Также одним из вариантов повышения эффективности направления США может стать перераспределение рекламного бюджета между каналами, например на каналы, которые показывают более высокие показатели эффективности: MediaTornado
2. Основной рекламный канал для Европейского рынка -  AdNonSence, который также как и TipTop и FaceBoom не окупается, ввиду чего представляется целесообразным перераспределить бюджет на более доходные каналы привлечения, такие как: lambdaMediaAds и WahooNetBanner, которые также показывают высокий уровень привлекаемых платящих пользователей
3. Кроме того, неравномерные данные на графиках удержания пользователей из Германии и Франции говорит о том, что возможно необходимо усилить рекламные мероприятия, направленные на преодоление оттока пользователей, «недоживающих» до окончания 2 недельного периода
4. Также, видится целесообразным, проработать вопросы, связанные с привлечением бОльшего количества пользователей РС, которые среди пользователей всех остальных устройств оказались с наилучшим уровнем удержания, но с наихудшим уровнем конвертации.
